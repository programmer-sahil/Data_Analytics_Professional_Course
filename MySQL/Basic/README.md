# MySQL Beginner Practice

This repository contains beginner-level MySQL practice done in MySQL Workbench.

## Summary
In this practice, we learned how to:
- create a database
- select a database
- create a table
- insert records
- display records using `SELECT`
- filter data using `WHERE`
- update records using `UPDATE`
- delete records using `DELETE`

We also understood the use of:
- rows and columns
- basic data types like `INT` and `VARCHAR`
- primary key concept
- safe update mode in MySQL Workbench

## Tools Used
- MySQL
- MySQL Workbench

## Purpose
This project is for beginner-level learning and practice of basic SQL commands.

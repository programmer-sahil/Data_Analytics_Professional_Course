# MySQL Class 2 Practice

This repository contains beginner-level MySQL practice completed in MySQL Workbench.

## Topics Covered
In this class, we practiced:

- creating a new database
- selecting a database using `USE`
- creating tables with constraints
- inserting records into tables
- viewing records using `SELECT`

## Constraints Learned
We worked with the following MySQL constraints:

- `PRIMARY KEY` – to uniquely identify each row
- `NOT NULL` – to prevent empty values
- `UNIQUE` – to avoid duplicate values
- `DEFAULT` – to set an automatic value
- `AUTO_INCREMENT` – to generate ids automatically

## Practice Tables Created
- `students`
- `learners`
- `users`
- `admission`
- `employees`
- `college_students`

## Summary
This class focused on understanding how constraints help maintain correct and organized data in MySQL tables. We also practiced inserting and displaying records using simple SQL queries.
